# Minecraft-Python-Projects
Small Minecraft-related projects I'm experimenting with. 👽👾

# All my projects are in the order they were made so that I can see my progress over time. 😎

1. Minecraft Farm Checker (v1) : 
Takes requirements for a farm and tells you if you have enough resources for a decent farm plus other thingalings 🥬🍒

2. Minecraft Mining Trip Checker (v1) : 
This evaluates whether a player has the resources, equipment, and supplies needed for a safe and efficient minecraft mining expedition ⛏💣

3. Nether Expedition Checker (v1) :
This evaluates gear, supplies, survival essentials and stuff before entering the nether ⚔️

5. End Expedition Checker (v1) :
My biggest project so far! It is a big multi-stage minecraft end guide covering strongholds, dragon fights, end cities, elytra hunting, survival tips and other shenanigans 🤭
